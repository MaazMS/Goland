## godoc  
1. The greatest thing about godoc and this is what you need to know is that you can run a local server and see all of    
   your documentation just right from a local service only.  
```bash
Open terminal
godoc -http  :8080  or
Godoc -http =:8080
Open browser
http://localhost:8080 
Ctrl +c kill service

```
