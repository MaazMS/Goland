## Conditional logic operators  
|Operator |	Meaning| 	Example
|--- | ---- | --- |
|and |	True if both the operands are true   |	x `&&` y
|or	 |True if either of the operands is true |	x `||` y
|not | 	True if operand is false (complements the operand) |	`!` x
