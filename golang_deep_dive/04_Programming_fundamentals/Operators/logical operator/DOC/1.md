### Logical Operators  
1. It is used to combine two or more conditions.  
2. It return true and false.  
* Syntax
'condition1 operator condition2`  
* Example  
```
    intno1 := 10
	intno2 := 20

	floatno1 := 10.0
	floatno2 := 20.0

	res1 := intno1 != intno2 && floatno1 != floatno2
	fmt.Println(res1) 
```     

name of operator| sign of operator|
--- |--- |
Logical AND | `&&`
Logical OR | `|| `
Logical NOT | `!`
